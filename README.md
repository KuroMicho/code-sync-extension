# CodeSync Classroom Client 📟

Sistema de sincronización y monitoreo de código en tiempo real diseñado para laboratorios de programación y entornos educativos. Permite la supervisión interactiva entre docentes y estudiantes de forma ligera y eficiente.

## 🚀 Características Principales

- **Sincronización en Vivo:** Monitoreo en tiempo real de los archivos de código de los estudiantes desde la vista del docente.
- **Modo Desafío (Timer):** Control de tiempos para exámenes y retos con alertas sonoras Lo-fi integradas (`.mp3` nativo).
- **Chat Cyber-Room:** Pestaña de comunicación interactiva con estética neón integrada directamente en el entorno de desarrollo.
- **Mano Virtual (Asistencia):** Sistema de alerta visual `$(question)` en el panel del profesor para soporte prioritario.
- **Entrega Automatizada:** Envío de snapshots completos del proyecto al finalizar el temporizador de forma segura.

## 🛠️ Instalación

1. Copie el archivo `.vsix` generado a la máquina de destino.
2. En VS Code, vaya a la sección de extensiones (`Ctrl + Shift + X`).
3. Haga clic en los tres puntos (`...`) de la esquina superior derecha y seleccione **Install from VSIX...**.
4. Seleccione el instalador de CodeSync.
